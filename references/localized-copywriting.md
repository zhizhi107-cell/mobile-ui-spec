# Localized UI Copywriting

Use natural, concise product language for the target locale. Avoid translated phrasing that does not sound native in the product's language.

## General Rules

- Use the product's target language by default.
- For Chinese products, use natural, concise Chinese product language.
- For English or international products, use concise English UI copy with familiar product terminology.
- Prefer short labels for tabs, buttons, and menus.
- Prefer verbs for action buttons.
- Keep error and empty-state copy brief but helpful.
- Avoid decorative slogans in operational interfaces.

## Common Labels

Use:
- Chinese: 保存, 提交, 确认, 取消, 编辑, 删除, 查看, 筛选, 排序, 搜索, 重试, 完成
- English: Save, Submit, Confirm, Cancel, Edit, Delete, View, Filter, Sort, Search, Retry, Done

Avoid stiff translations:
- "应用过滤器" -> "筛选"
- "执行搜索" -> "搜索"
- "提交表单" -> "提交"
- "移除项目" -> "删除"
- "Perform search" -> "Search"
- "Apply filters" -> "Filter" or "Apply"
- "Submit form" -> "Submit"
- "Remove item" -> "Delete" or "Remove", depending on risk and reversibility

## Empty States

Good examples:
- Chinese: 暂无数据, 还没有记录, 未找到结果, 暂无消息, 暂无可用内容
- English: No data yet, No records yet, No results found, No messages, No available content

When useful, add one action:
- Chinese: 暂无记录，去新建 / 未找到结果，换个关键词试试
- English: No records yet. Create one / No results found. Try another keyword.

## Error States

Good examples:
- Chinese: 网络异常，请稍后重试 / 保存失败，请重试 / 内容加载失败 / 请输入手机号 / 验证码错误，请重新输入
- English: Network error. Try again later. / Save failed. Try again. / Failed to load content. / Enter a phone number. / Incorrect code. Try again.

## Length Guidance

- Chinese tab labels: usually 2-4 Chinese characters.
- Chinese primary buttons: usually 2-6 Chinese characters.
- English tab labels: usually 1-2 short words.
- English primary buttons: usually 1-3 words.
- List titles: allow one or two lines depending on content importance.
- Secondary metadata: short, scannable, and lower emphasis.

## Tone By Domain

- Tools and productivity: direct, calm, efficient.
- Commerce: clear and action-oriented, not noisy.
- Content and community: friendly but not wordy.
- Finance, healthcare, government: precise, trustworthy, low ambiguity.
